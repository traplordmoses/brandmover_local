DETERMINISTIC TEMPLATE SYSTEM

NEW IN v8 — PIL RENDERER + FIGMA IMPORT
THE PROBLEM AND THE FIX
Templates Without AI Drift
Before: Template frames were generated via flux-kontext-pro at strength=0.75. Diffusion models approximate — they don't replicate pixel-perfectly. Small details drifted each generation.

Now: Claude Vision extracts a precise visual spec from a reference image → PIL renders the template frame deterministically (exact colors, shapes, positions) → users refine via conversation with instant re-renders. Zero API cost for template iteration.
PATH 1: UPLOAD A REFERENCE IMAGE
1. /template_from_reference
Upload any template image as reference
↓
2. Claude Vision Analysis
Extracts TemplateSpec JSON: background type/colors, shapes with positions/borders/radii, text zones with font estimates, image zones
↓
3. PIL Renders Preview
Instant render — checkerboard behind transparent zones, labeled overlays (IMAGE 1, TEXT: title, LOGO). No API call.
↓
4. Conversational Refinement
"Make the background darker" → spec modified → instant re-render. No Replicate cost. Iterate as many times as needed.
↓
5. Save Template
Saved to template manifest with spec_json for future re-rendering after brand color changes
PATH 2: IMPORT FROM FIGMA
1. /template_import <figma_url>

↓
4. Template Ready
Screenshot used as-is (pixel-perfect from Figma). Regions mapped from classified layers. Same preview flow.

SECURITY HARDENING & PERFORMANCE

FULL AUDIT — v8
SECURITY FIXES
Path Traversal Prevention
All Telegram file uploads (fonts, PDFs) stripped with os.path.basename(). No ../../../etc/passwd attacks possible.
img2img Path Validation
File reads restricted to brand/ and /tmp directories. Image extension whitelist (.png, .jpg, .jpeg, .webp, .gif).
File Size Limits
Fonts: 10MB max. PDFs: 50MB max. Validated with isinstance(int) check before comparison.
Gemini API Key in Header
Moved from URL query parameter to x-goog-api-key header. No longer visible in logs or browser history.
OpenClaw Newline Bypass
Added \n\r\t to unsafe chars regex. Prevents command injection via newline characters in scripts.
Generic Error Messages
Removed raw str(e) from user-facing messages. Internal errors no longer leak stack traces to Telegram.
PERFORMANCE OPTIMIZATIONS
Singleton API Clients
agent/_client.py — one AsyncAnthropic and one httpx.AsyncClient reused across all 35 modules. Previously: new client per call (24+ instantiation sites). Connection pooling saves ~100ms per LLM call.
NumPy Vectorized Gradients
Template renderer gradients use np.meshgrid + np.linspace + Image.fromarray instead of per-pixel putpixel(). ~100x faster for large gradient fills.
Cached Compositor Backgrounds
Background images + fade masks cached by config hash + dimensions. Avoids re-rendering 14 layers per compose for repeated generations.
Parallel Multi-Option Compose
brand_3d 3-option images composed via asyncio.gather() instead of sequential loop. ~3x faster for multi-option drafts.
Async Font Downloads
Google Fonts fetched with async httpx (non-blocking). Cached to brand/assets/fonts/ after first download.
State Concurrency Safety
asyncio.Lock on state file operations. In-memory cache reduces repeated JSON parsing. Prevents race conditions between auto-poster and user handlers.
